char _newrelic_process_config_t::daemon_socket[512]

Optional. Specifies the underlying communication mechanism for the agent daemon. The default is to use a UNIX-domain socket located at /tmp/.newrelic.sock. If you want to use UNIX domain sockets then this value must begin with a "/". Setting this to an integer value in the range 1-65534 will instruct the daemon to listen on a normal TCP socket on the port specified. On Linux, an abstract socket can be created by prefixing the socket name with '@'.
